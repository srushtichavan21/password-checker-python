# 🔐 Password Strength Checker (Python)

A simple Python tool to evaluate the strength of a password based on security rules.

---

## 🚀 Features

* Checks password length
* Detects uppercase & lowercase letters
* Validates numbers and special characters
* Classifies password as Weak / Medium / Strong

---

## 🛠 Tech Used

* Python
* Regular Expressions (`re` module)

---

## ▶️ How to Run

```bash
python main.py
```

---

## 💡 Example

Input: `Hello123`
Output: `Medium ⚠️`
